## Getting Started üöÄ

### Prerequisites

- Java 11 or higher
- SBT (Scala Build Tool)
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/fadismoqyS/colorwoodSort-25.git
cd colorwoodSort-25
```

2. Build the project:
```bash
sbt compile
```

### Usage

- Run the application:
```bash
sbt run
```

- Start Scala REPL:
```bash
sbt console
```

- Run tests:
```bash
sbt test
```
